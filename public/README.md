# Static assets

Drop these two files here:

- `profile.jpg` — your headshot/photo (any size; the frame is 260×320 and uses `object-fit: cover`, so a portrait crop works best)
- `resume.pdf` — your resume; the "Resume" link in the contact section opens it in a new tab

If `profile.jpg` is missing the hero photo card falls back to an emoji + a note.
